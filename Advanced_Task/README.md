
# Advanced Task – Spam Detection using Machine Learning

## 📌 Description
In this task, I developed a Spam Detection System using Machine Learning in Python. The system classifies text messages as spam or ham (not spam) based on their content.

## 🛠️ Tools Used
- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Multinomial Naive Bayes

## 📊 Work Done
- Created a dataset (spam.csv) with labeled messages
- Performed text preprocessing (converted text to lowercase)
- Converted text data into numerical format using CountVectorizer
- Split the dataset into training and testing sets
- Trained the model using Multinomial Naive Bayes
- Tested the model on unseen data
- Calculated accuracy of the model
- Generated classification report
- Predicted results for sample messages

## 🎯 Outcome
This task helped me understand how machine learning models can be used for text classification problems like spam detection. I also learned about model training, testing, and evaluation.
