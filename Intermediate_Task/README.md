
# Intermediate Task – Sentiment Analysis using Python

## 📌 Description
In this task, I developed a Sentiment Analysis system using Python. The program analyzes text data and classifies it as positive or negative. It also visualizes the results using a graph.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Work Done
- Created a dataset of sample sentences
- Classified sentences as positive or negative
- Counted the number of positive and negative sentiments
- Displayed the results in the terminal
- Generated a graph to visualize sentiment distribution

## 🎯 Outcome
This task helped me understand how text data can be analyzed and visualized using Python.
