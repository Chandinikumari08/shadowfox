# ShadowFox Internship – Data Science & Machine Learning Tasks

## 📌 Overview
This repository contains all the tasks completed during my ShadowFox Internship. The tasks are divided into three levels: Beginner, Intermediate, and Advanced. Each level focuses on different concepts of Python, Data Analysis, and Machine Learning.

---

## 🥇 Beginner Task – Data Visualization
In this task, I worked on data visualization using Python. I created different types of graphs to understand and represent data visually.

### 🔹 Key Work:
- Line Plot
- Bar Chart
- Histogram
- Scatter Plot
- Pie Chart

### 🛠️ Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 🥈 Intermediate Task – Sentiment Analysis
In this task, I developed a Sentiment Analysis system that classifies text as positive or negative. I also visualized the sentiment distribution using a graph.

### 🔹 Key Work:
- Text classification (Positive/Negative)
- Sentiment counting
- Graph visualization

### 🛠️ Tools Used:
- Python
- Pandas
- Matplotlib

---

## 🥉 Advanced Task – Spam Detection System
In this task, I built a Machine Learning model to detect spam messages. The model classifies messages as spam or ham.

### 🔹 Key Work:
- Data preprocessing
- Feature extraction using CountVectorizer
- Train-test split
- Model training using Naive Bayes
- Accuracy and classification report
- Predictions on new messages

### 🛠️ Tools Used:
- Python
- Pandas
- Scikit-learn
